# T1_Hoyos_Yrigoyn

##Funcionalidades
- Registro de estudiantes con 3 notas.
- Cálculo automático de promedio.
- Evaluación condicional (Aprobado/Desaprobado).
- Clasificación por rendimiento.
- Menú interactivo con ciclo `while`.
- Control de versiones con Git (ramas, merges, conflictos).

##Tecnología
- C# (.NET)
- Git & GitHub

##Estructura
- `/CSharp`: Código en C#.

  
##Este proyecto demuestra el dominio de:
- Estructuras secuenciales, condicionales y repetitivas.
- Uso de operadores y tipos de datos.
- Control de versiones con flujos de trabajo, ramas y resolución de conflictos.
